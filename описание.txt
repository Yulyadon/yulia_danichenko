Приложение созданно при помощи: HTML, CSS и JavaScript
Макет приложения сделан с помощью использования grid. Реализована адаптивность.
В шапке приложения закреплено меню, с его помощью можно перемещаться по основным частям страницы приложения.
Далее следует поиск по книгам. В результате поиска отображается: название книги, ФИО автора, жанр и количество страниц книги.
Следующая часть это редактирование и создание авторов/жанров/книг. Можно добавить новые элементы при нажатии на кнопку "добавить..." и редактировать элементы выбрав их из выпадающего списка.

Реализована валидация, сохранение происходит только если поля ввода проходят валидацию.
В localStorage создан объект "db" с массивами объектов. В него сохраняются все жанры, книги, авторы, а так же оттуда выводятся в другие части приложения.
